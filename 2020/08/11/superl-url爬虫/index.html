Superl-url是根据关键词，对搜索引擎内容检索结果的网址内容进行采集的一款轻量级软程序

项目地址：

​ github https://github.com/super-l/superl-url

距离作者上一次更新也挺久的了，所以搜索引擎都失效了，需要手动修改

四、其他遇到的问题

运行的时候发现报错了

应该是https的关系

添加 verify=False 取消https验证

response = requests.get(url, verify=False) # 参数：verify：Ture/False，默认是Ture，用于验证SSL证书开关

后来发现报错，识别不了
