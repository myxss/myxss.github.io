一、pyspider安装

​ 项目地址

​ https://github.com/binux/pyspider

​ 直接官方教程

​ http://docs.pyspider.org/en/latest/

​ 本机用的是ubuntu,先安装依赖环境

​ apt-get install python python-dev python-distribute python-pip \ libcurl4-openssl-dev libxml2-dev libxslt1-dev python-lxml \ libssl-dev zlib1g-dev

​ 安装python3 、 pip3 （这里需要注意python3.7版本中有一个变量名冲突，会报错，如果嫌麻烦，可以降级使用3.6）

​ pip3 install pyspider

​

​ 同时请根据需要安装PhantomJS：http://phantomjs.org/build.html（请添加到环境变量）
